# Alumora

Alumora is a Django-based alumni-student networking platform designed to connect students, alumni, and institutions.

## Features

- Student Dashboard
- Alumni Dashboard
- User Authentication
- Editable Profiles
- Cover Photo & Profile Photo
- Responsive UI
- Role-based Dashboards

## Tech Stack

- Django
- Python
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

# Installation

```bash
git clone https://github.com/palak-pandey/Alumora.git
cd Alumora
pip install -r requirements.txt
python manage.py runserver
